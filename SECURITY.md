# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public issue
2. Email security concerns to: security@sanar.dev
3. Include detailed steps to reproduce the vulnerability
4. Allow reasonable time for a fix before public disclosure

## Supported Versions

| Project | Supported Versions |
|---------|-------------------|
| sanar   | Latest release    |
| forge   | Latest release    |
| axon    | Latest release    |

## Security Updates

Security patches are released as soon as possible after verification. Subscribe to repository releases to be notified.
